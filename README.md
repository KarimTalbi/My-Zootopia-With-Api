# My Zootopia

This project fetches animal data from an API and generates an HTML file to display it. (part of the Masterschool curriculum)

## How to Use

1.  Install the dependencies:
    
    ```
    pip install -r requirements.txt
    ```
    
2.  Run the `animals_web_generator.py` file:
    
    ```
    python animals_web_generator.py
    ```
    
3.  Follow the prompts to enter an animal name and select a filter.
4.  An `animals.html` file will be generated in the project's root directory.
